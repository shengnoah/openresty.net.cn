### 说明

- API Server默认在7777端口监听
- 若无特别说明，API均以HTTP协议给出，返回值为json格式，数据格式如下:

```
#失败情况
{
    success: false,
    msg: "错误描述"
}

#成功情况
{
    success: true,
    msg: "描述信息",
    data: {}//相关数据
}
```

### 插件基本信息API

URI                 | Method 
------------------- | ---- 
/plugins            | Get 

**参数** 
无 

**返回结果**
